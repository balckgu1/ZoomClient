# Test Playbook

When asked to test skill function, walk through the following checklist:

1. - Generate a test.txt file with the following content: `Hello skill. This is a test file.`
2. - Generate a test.py file with the content `print" ("Hello skill. This is a test python file")`
3. - Add a paragraph at the end of test. txt: `This's right.`
